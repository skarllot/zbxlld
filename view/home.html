ZBXLLD

Provides additional discovery features to Zabbix-Agent

View project on GitHub »

Why ZBXLLD?

Official Zabbix-Agent executable lacks detailed discovery for drives, network and services. ZBXLLD fill that void through user parameters to Zabbix-Agent.

Easy Install

To install ZBXLLD you just need to follow the three steps below:

  • Download and decompress ZBXLLD executable to any directory

  • Add the line indicated below to zabbix_agentd.conf:

    UserParameter=zbxlld[*],C:\Zabbix\bin\zbxlld-win.exe $1 $2 $3
  • Restart Zabbix-Agent service.
